# Release Notes - Sanaei X-UI v2.8.5 Support

## New Features
- **Sanaei X-UI v2.8.5 Support**: Full compatibility with the latest version of Sanaei X-UI panel.
- **Backward Compatibility**: Automatically detects and supports older versions (v1.7.9) without manual configuration.
- **Smart API Handling**: Implemented intelligent API endpoint switching with automatic fallback.

## Changes
- Updated `config.php` to include new API helper functions.
- Modified all API calls to use the new `/panel/api/inbounds/` format.
- Updated documentation (`README.md` and `README-fa.md`) to reflect new version support.

## Installation/Update
To update your existing bot, simply pull the latest changes. No database or configuration file changes are required.
